Choisir L'editions


Editions Standard :
les joueurs qui achètent l'édition standard de Arsernal pourraient recevoir un code pour télécharger un pack d'armes exclusives qui ne sont pas disponibles dans le jeu de base.
Editions Prenium :
Jeu de base + une version physique de la bande-son en bonus, livrée sous forme de CD ou de vinyle. des skins exclusifs pour les personnages, les armes ou les véhicules, qui ne sont pas disponibles dans le jeu de base.
Editions Gold :
Une bande-son physique des skins exclusifs pour les personnages, les armes ou les véhicules, qui ne sont pas disponibles dans le jeu de base. pour récompenser les joueurs qui achètent l'édition Gold, vous pourriez profiter d'un accès anticipé à tous les DLC qui seront publiés pour le jeu, avant leur sortie officielle.
